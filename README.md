# Heart Disease Prediction Web App

This project is a web-based application that predicts the risk of heart disease
using a Machine Learning Random Forest model.

The application is built using Python and Streamlit.

---

## Features

- Patient health input form
- Heart disease risk prediction
- Risk percentage display
- Interactive charts and graphs
- Feature importance visualization
- Dark themed UI

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
- Joblib

---

## Files in This Project

heart.py               -> Streamlit web application  
best_rf_model.pkl      -> Trained Random Forest model  
heart.csv              -> Dataset  
requirements.txt       -> Required Python packages  
README.md              -> Project explanation  

---

## How to Run This Project

1. Open terminal inside the project folder

2. Install required packages:
pip install -r requirements.txt

3. Run the Streamlit app:
streamlit run heart.py

---

## Output

- Displays heart disease risk (Low / High)
- Shows risk percentage
- Shows patient health graphs
- Shows model feature importance graph
